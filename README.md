# Эмулятор электросчётчиков с интерфейсом RS-485
Проект написан на Delphi.
На данный эмулируются следующие модели:
- Инкотекс-СК "Меркурий 206"
- Энергомера "СЕ102"
- Энергомера "СЕ102М"
- IEK "Star 104/1"
